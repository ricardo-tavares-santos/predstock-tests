# predstock-tests

predstock tests

## Instructions

**1. Clone this repo**

```
git clone https://github.com/ricardo-tavares-santos/predstock-tests.git
```

**2. NPM install React and dependencies**

```
npm install
```

**3. Start**

```
npm start
```

**4. listens on port 3000**

```
http://localhost:3000/AAPL
```
